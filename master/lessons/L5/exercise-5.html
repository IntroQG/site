Exercise 5¶

Warning

Please note that we provide assignment feedback only for students enrolled in the course at the University of Helsinki.

Start your assignment

You can start working on your copy of Exercise 5 by accepting the GitHub Classroom assignment.

Exercise 5 is due by the start of lecture in week 6.

You can also take a look at the open course copy of Exercise 5 in the course GitHub repository (does not require logging in). Note that you should not try to make changes to this copy of the exercise, but rather only to the copy available via GitHub Classroom.

Hints for Exercise 5¶

P1: Solving for the unknown pressures in Equation 10¶

In Part 2 of Problem 1 you’re asked to solve for the term \((p_{1} - p_{0}) / L\) in Equation 10. To help you know whether or not you’ve done this properly, I’ve given a table of the values you should get below.

\(n\) \((p_{1} - p_{0}) / L\)
2 4215743440.23
3 8029987.50521
4 14339.2634022
5 24.5815944037

Solving for the unknown term in Equation 19¶

Similar to Problem 2, you’re again asked to solve for an unknown term in Equation 19, in this case \(\gamma_{x}\). As above, you have a table of values below to help you check whether or not you’ve properly solved your equation for \(\gamma_{x}\).
